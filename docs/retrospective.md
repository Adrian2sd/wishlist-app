# Reflexión Final

## Aprendizajes
- Conectar frontend React con un backend Express y tipar las respuestas.
- Gestionar estado global con Context + hooks.
- Aplicar arquitectura por capas en el backend.

## Problemas encontrados
- Configuración de CORS para desarrollo local.
- Sincronización de tipos entre frontend y backend.
- Manejar promesas en el cliente tipado.

## Uso de IA
La IA ayudó a generar el esqueleto de la API, la configuración de Tailwind y la documentación, reduciendo el tiempo de desarrollo y proporcionando ejemplos claros.

## Conclusión
El proyecto demostró cómo una arquitectura bien definida facilita el desarrollo fullstack, manteniendo la separación de responsabilidades y la tipificación estática.
